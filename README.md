# OpenClaw 部署 Skill

从零部署 OpenClaw AI 聊天伴侣的完整指南：本地配置 → 微信接入 → 记忆系统 → 身份塑造 → 云服务器上线。

## 内容

- [搭建全记录.md](./搭建全记录.md) — 📘 完整搭建历程、架构决策、踩坑记录
- [openclaw-weixin-skill.md](./openclaw-weixin-skill.md) — OpenClaw 微信聊天伴侣完整部署文档

## 适用环境

- Windows 11 本地开发
- 腾讯云轻量应用服务器 (OpenCloudOS 9.4)
- DeepSeek API (v4-pro / v4-flash)
- 微信接入 (openclaw-weixin 插件)
